Squirrel how-to...

Undo a change

You can undo changes made locally, using . You cannot undo changes made in the cloud.

Change the text size

Click/tap and use the and buttons in the dialog.

Add a new folder

Add new top-level folders by right-clicking (or long press) on the topmost and selecting Add new folder from the pop-up menu. Enter a name for the folder, , and the folder will be created and opened.

You can now use the pop-up menu on the new folder to add sub-folders and values.

Edit a name or value

Either select Rename or Edit Value from the menu (right-click or long press), or double-click/tap on a name or a value to edit it

Copy a value

You can use the Copy Value menu item, or simply double click/tap on the value and use the system clip function (e.g. Ctrl+C on desktops)

Search

Enter the text you want to search for into the Search: box at the top. , then click/press

Search hits will be highlit and scrolled to. You can scroll through hits using

Both keys and values are searched, and case doesn't matter.

Move a folder or value to another folder

Press and hold the right-hand button for the folder or value you want to move, and drag it to its new home. It will be moved when you lift off.

Change the UI Language

Click and enter the ISO 639-1 code for the language in Language:. The UI will change to your chosen language (if it is available!) when you close the dialog.

For help adding a language, visit the Squirrel github site.

Export my data

Export a folder or value by selecting Copy A text version (in JSON format) will be copied to the clipboard.

Text in JSON format on the clipboard can be pasted into a new folder using Paste

So you can copy a folder, paste it into a text editor, modify it there, and then paste the result back into Squirrel.
